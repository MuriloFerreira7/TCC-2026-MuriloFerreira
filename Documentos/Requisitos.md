# Requisitos
| ID   | Nome do Requisito                   | Descrição                                                                                                     |
| ---- | ----------------------------------- | ------------------------------------------------------------------------------------------------------------- |
| RF01 | Cadastro de hábitos                 | O sistema deve permitir que o usuário crie novos hábitos personalizados.                                      |
| RF02 | Sugestão de hábitos iniciais        | O sistema deve sugerir hábitos simples no início do uso.                                                      |
| RF03 | Progressão de dificuldade           | O sistema deve aumentar gradualmente a dificuldade e quantidade de hábitos com base no desempenho do usuário. |
| RF04 | Tipos de hábitos                    | O sistema deve permitir diferentes tipos de hábitos (ex: contador, simples).                                  |
| RF05 | Definição de frequência             | O sistema deve permitir definir a frequência dos hábitos (diário, semanal, quinzenal ou único).               |
| RF06 | Hábitos pré-definidos               | O sistema deve oferecer uma lista de hábitos genéricos editáveis.                                             |
| RF07 | Personalização de hábitos           | O sistema deve permitir personalizar hábitos, incluindo valores, unidades e parâmetros.                       |
| RF08 | Categorização de hábitos            | O sistema deve organizar hábitos em categorias (saúde, esportes, lazer, finanças, etc.).                      |
| RF09 | Edição de hábitos                   | O sistema deve permitir editar nome, tipo e frequência dos hábitos.                                           |
| RF10 | Exclusão automática de hábito único | O sistema deve remover hábitos definidos como únicos após sua conclusão.                                      |
| RF11 | Registro de progresso               | O sistema deve registrar a conclusão dos hábitos ao longo do tempo.                                           |
| RF12 | Visualização de desempenho geral    | O sistema deve exibir a porcentagem de hábitos concluídos por período.                                        |
| RF13 | Detalhamento de desempenho          | O sistema deve permitir visualizar quais hábitos foram concluídos ou não em um período específico.            |
| RF14 | Relatórios periódicos               | O sistema deve permitir a criação de relatórios diários, semanais e mensais.                                  |
| RF15 | Registro de relatos                 | O sistema deve permitir que o usuário escreva relatos sobre sua experiência em cada período.                  |
| RF16 | Escala de satisfação                | O sistema deve permitir que o usuário avalie seu nível de satisfação ao final de cada período.                |
| RF17 | Sistema de notificações             | O sistema deve enviar lembretes para a execução dos hábitos planejados.                                       |
| RF18 | Organização de rotinas              | O sistema deve organizar os hábitos em uma estrutura de rotina diária para facilitar sua execução.            |
